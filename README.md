# Thunderbolt Fighter

[简体中文](README.zh-CN.md)

Thunderbolt Fighter is an offline desktop arcade shooter built with Electron and Phaser. Choose a run length and difficulty, dodge enemy fire, collect pickups, chase local best scores, and defeat the boss before the sortie ends.

## Prerequisites

Install Node.js with npm before running or building from source. The release workflow uses Node.js 22, so that version is the safest match for local packaging.

## Install

```sh
npm install
```

## Run Locally

```sh
npm run dev
```

## Build The Windows App

```sh
npm run package:win
```

## Download A Prebuilt Release

Players who do not want to install Node.js can download the prebuilt Windows zip, `thunderbolt-fighter-win32-x64.zip`, from the project's GitHub Releases page: https://github.com/Authorm-lh/Thunderbolt-Fighter/releases
